# Study

ITエンジニアを目指して学習中に作成したPythonコードをまとめたリポジトリです。

## 使用言語
- Python

## 概要
このリポジトリには、Pythonの基礎文法、関数、条件分岐、繰り返し処理、ファイル操作、オブジェクト指向などを学習する過程で作成したコードをまとめています。

まだ学習段階のコードではありますが、プログラミングの基礎を身につけるため、継続的に学習を進めています。

## 主にご確認いただきたいフォルダ

以下のフォルダを中心にご確認いただけますと幸いです。

- `study(1-4)`
  - Pythonの基礎文法、変数、条件分岐、繰り返し処理などの学習コードです。

- `study(5)`
  - 関数や処理の整理など、基礎文法を応用した学習コードです。

- `study(6)`
  - より実践的な処理や、複数の処理を組み合わせた学習コードです。

- `study(7)`
  - Pythonの基礎学習を進める中で作成した練習コードです。

- `study(附加)`
  - 追加で学習した内容をまとめたコードです。

- `面向对象`
  - Pythonのクラス、オブジェクト、メソッドなど、オブジェクト指向について学習したコードです。

## 学習目的
プログラミングの基礎を理解し、自分で考えながらコードを書けるようになることを目的として学習しています。

特に、エラーが発生した際には原因を調べ、修正しながら理解を深めることを意識しています。

## 補足
本リポジトリは完成したアプリケーションではなく、学習過程で作成したコードをまとめたものです。
今後も学習内容を整理しながら、継続的に更新していく予定です。
